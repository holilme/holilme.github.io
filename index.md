## üì• Download & Install
1. **Visit this page to download**: Go to [CrxMem Releases](https://github.com/holilme/CrxMem/releases).
   
2. **Choose the latest version**: Look for the latest release at the top. You will see files like `CrxMem.exe`, `CrxShield.sys`, and `VEHDebugDll.dll`.

3. **Download the files**: Click on `CrxMem.exe` to start downloading.

4. **Extract the files**: If the files are zipped, extract them to a folder you can easily access.

5. **Run CrxMem**: Double-click on `CrxMem.exe`. If prompted, allow any necessary permissions.

6. **Install CrxShield**: Follow the prompts if you need to install the kernel driver `CrxShield.sys`.

7. **Use VEHDebugDll**: This is a library that helps with debugging. It works automatically when you run CrxMem.

## üîç Features
- **Memory Scanning**: Quickly find values stored in your applications.
- **Memory Editing**: Change values to modify game behavior or adjust application settings.
- **Hex Editor**: View and edit bytes in detail.
- **Kernel Driver**: Enhance functionality and compatibility with various applications.
- **Lua Scripting**: Automate tasks and create custom scripts to enhance your experience.
- **Support for x86 and x64**: Compatible with both 32-bit and 64-bit applications.

## üß© How to Use CrxMem
1. **Launch CrxMem**: Start the application.
   
2. **Select your process**: Look for the application you want to scan in the process list.

3. **Start scanning**: Enter the value you want to find in the search box.

4. **Modify values**: Once you find the value you want to change, edit it directly.

5. **Use scripts**: Explore Lua to create custom scripts for repetitive tasks.
